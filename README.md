# 🚀 Support Vector Classifier (SVC) Implementation

An end-to-end Machine Learning project demonstrating the implementation of **Support Vector Classifier (SVC)** using Python and Scikit-learn for binary classification. The project compares different SVM kernels and performs hyperparameter tuning to optimize model performance.

---

## 📖 Project Overview

This project explores the Support Vector Machine (SVM) algorithm for binary classification using a synthetically generated dataset. It demonstrates how different kernel functions affect classification performance and how GridSearchCV can be used to find the best hyperparameters.

---

## ✨ Key Features

- 📌 Synthetic dataset generation using `make_classification()`
- 📌 Data visualization with Seaborn
- 📌 Train-test data splitting
- 📌 Support Vector Classifier (SVC) implementation
- 📌 Comparison of multiple kernels:
  - Linear
  - RBF (Radial Basis Function)
  - Polynomial
  - Sigmoid
- 📌 Hyperparameter tuning using GridSearchCV
- 📌 Model evaluation using:
  - Classification Report
  - Confusion Matrix
